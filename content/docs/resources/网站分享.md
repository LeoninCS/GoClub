# 网站分享

这里整理一些适合日常学习、专题查阅和长期收藏的网站。

## 技术专栏与课程索引

### 梁亮乐的学习站

这个站点更像一个技术资料导航站，集中整理了不少技术专栏、课程笔记和文章索引。里面能看到很多常见主题，比如 Go、MySQL、Redis、Kubernetes、操作系统、架构设计和面试相关内容，比较适合在学习过程中按专题快速查资料。

- 适合场景：按专题找资料、查课程索引、做阶段性补充学习
- 覆盖方向：后端开发、基础知识、工程实践、面试准备
- 访问地址：https://learn.lianglianglee.com/

### Linux.do 社区

这是一个高活跃度的技术社区，内容覆盖开发实践、工具讨论、踩坑排查和经验分享。整体交流质量和信息时效性都不错，这里也认可它作为日常学习与问题检索的参考社区。

- 适合场景：问题检索、技术讨论、工具经验交流
- 覆盖方向：开发实践、效率工具、工程经验、AI 相关应用
- 访问地址：https://linux.do/
