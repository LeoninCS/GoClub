### 6. 了解浏览器的渲染流程吗？

答：浏览器先做 DNS 解析，再建立 TCP/TLS 连接，发送 HTTP 请求，服务端返回资源，浏览器解析 HTML、CSS、JS，构建 DOM/CSSOM 并完成布局绘制。

### 7. HTTPS 的 S 讲一下

答：HTTPS 是 HTTP 运行在 TLS 之上。TLS 通过证书校验身份，协商会话密钥，后续使用对称加密保护传输内容。

### 8. 了解鉴权吗？token 存在哪里？还有其他方式吗？

答：鉴权链路通常是登录签发 token，请求时中间件校验签名、过期时间和用户状态。高安全场景会结合 Redis 黑名单、token 版本和刷新机制。

### 9. 了解 HTTP 的缓存吗？

答：网络题按分层回答：DNS、TCP/TLS、HTTP 请求响应、连接复用、拥塞控制和可靠传输。每个环节都能对应排障工具。
