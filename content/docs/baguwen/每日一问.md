## 2026.04.09

### MySQL 索引失效的场景？

1. 不满足最左匹配原则。
2. 对索引列做了函数操作。
3. 对索引列做了运算。
4. `SELECT *` 查询。
5. `LIKE '%xxx'` 前置通配符。
6. `OR` 条件使用不当。
7. 索引列和查询条件类型不一致，发生隐式转换。
8. 联合索引中，范围查询后面的列失效。

### Redis 的持久化机制？
